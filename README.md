# lidar-classification

## Описание
lidar-classification - это программа для определения типа атмосферных частиц на основе измерений их оптических свойств. В данном проекте мы используем данные измерений аэрозольной деполяризации, емкости флуоресценции и коэффициента преломления для классификации атмосферных частиц.

[Подробная документация](https://deepwiki.com/physicist2018/lidar-classification)
## Входные данные
Для расчетов нам потребуется три файла:

- `dep.txt` - файл с коэффициентами аэрозольной деполяризации
- `FL_cap.txt` - файл с коэффициентами емкости флуоресценции
- `mre.txt` - файл с данными о коэффициенте преломления

каждый файл должен содержать таблицу вида:

`dep.txt` 
```txt
alt-time  1    2
100     17.0  19.0
200      NaN  19.0
300     17.0  19.0
```
Первый столбец - высота в метрах
Первая строка - время

Ячейка на пересечерии первой строки и первого столбца "alt-time" - наимеенование столбцов
остальные ячейки - значения коэффициента деполяризации для соответствующей высоты и времени, представоенное в процентах.

Если данные за какую-то точку отсутствуют, в ячейке должно стоять `NaN`

Остальные файлы имеют схожее форматирование.
